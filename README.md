GifTastic!

-----------------------------

This application loads Gifs when a button is clicked. You can add more buttons by filling in the form. 
Clicking the gif will animate it, reclicking it will freeze the gif again.

-----------------------------

Made with:

Jquery

HTML

CSS

JavaScript

AJAX

Giphy API

-----------------------------
